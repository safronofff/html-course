Тег "form" - это блочный тег, предназначенный для передачи данных".
Сам он данные принять не может.
Атрибут "action" должен быть всегда заполнен.
Так же есть атрибут "method".
По умолчанию онн имеет значение "get" - что означает передачу данных открытым для пользователя способом.
Но таким способом объем данных меньше чем при значении "post",
который в свою очередь передает данные скрытым от пользователя способом.
Так же имеется атрибут "enctype" со значением "multipart/form-data".

Строчный тег, предназначенный для передачи данных.
В отличии от тега "form" он принемает данные от пользователя.
У него есть 3 атрибута:
"type" - тип данных ввода, по умолчанию стоит "text", что определяет ввод текста,
"name" - имя тега input,
"value" - значение тега, которое будет отображаться.

"checked" - атрибут выбора по умолчанию.

Для тегов "input" и "textarea" так же применяется атрибут "placeholder"
Он предназначен для подписи окна.

Для всех тегов, кроме тега "form" можно применить атрибут "disabled",
который блокирует все действия пользователей и делает пое тусклым(не активным).

Так же для всех тегов применяется атрибут "tabindex",
который определяет порядок перехода между полями.
У одной группы атрибутов "tabindex" должен иметь одно значение.

Тег "Textarea" предназначен для ввода текста.
Атрибуты "cols" и "rows" предназначены для определения размеров поля для вводимого текста.
Но как правило эти атрибуты не нужны, так как размер окна контролируется CSS и JS.

Для тегов "input" и "textarea" так же применяется атрибут "placeholder"
Он предназначен для подписи окна.

Выподающий список состоит из 2 тегов:
-тега "select", который является формой списка;
-и тега "option", который является пунктоми списка.
Атрибут "id" не нужен.


Иногда к выпадающему списку применяется атрибут "multiple".
Это нужно для множественного выбора.
Атрибут "selected" делает выбор по умолчанию.

Атрибут "button" предназначен для отправки.
Атрибут "type" имеет следующие значения:
-"submit" - предназначеный для отправки данных;
-"reset" - предназначенный для очистки данных.
